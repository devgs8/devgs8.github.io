## 📝 Hello World — Passo a Passo

### 1. Escreva o código

Crie um arquivo chamado `HelloWorld.java` com o seguinte conteúdo:

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
