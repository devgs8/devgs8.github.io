## 🚀 Why Java?

Java is one of the most popular programming languages in the world, and for good reason. It's **object-oriented**, **platform-independent** (thanks to the JVM), and **widely used** in enterprise applications, Android development, and backend systems.

### 💼 My Journey: From SysAdmin to Developer

I'm currently working as a **System Administrator**, but I've decided it's time for a change. I'm pursuing a degree in **Systems Analysis and Development**, and Java is a fundamental part of my curriculum — especially in key subjects like **Object-Oriented Programming** and **Algorithms**.

My goal is simple: **learn Java deeply and share everything I discover along the way**. This blog is my study journal, my accountability partner, and hopefully a resource for others on a similar path.

Why Java specifically?
- It's required for my university coursework
- It teaches essential programming concepts that transfer to any language
- It opens doors to backend development and enterprise careers

## 📦 Installing the JDK

First things first — you need the **Java Development Kit (JDK)**.

1. Go to [Oracle JDK](https://www.oracle.com/java/technologies/downloads/) or use **OpenJDK**
2. Download the version for your OS (I recommend Java 17 LTS)
3. Install and verify with the terminal:

```bash
java -version
